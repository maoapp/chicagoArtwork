## setup
This project was bootstrapped with Create React Native CLI.

Below you'll find information about performing common tasks.

## Install Dependencies

- yarn

## run server

- yarn start

## Running your React Native application

- yarn android
- yarn ios

## Generate android build

- yarn android:build
